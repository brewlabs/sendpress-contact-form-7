== Description ==

SendPress is a free newsletter plugin for WordPress that makes it really simple to send out email newsletters to your subscription lists. This plugin integrates Contact Form 7 with SendPress by providing an option for your customers to signup for your newsletter lists while submitting a form.

Support requests should go in the support forum or contact us at [SendPress.com](https://sendpress.com)

= Features =

* Allow your users to sign up for a SendPress newsletter list using a Contact Form 7 form
* You can capture first name, last name, and (of course) email
* You can signup users to as many lists as you like


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `sendpress-contact-form-7` directory to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

= Form Setup =

After installing & activating the plugin it's time to set up your form.

1. Click on Contact in the WordPress admin
1. Edit an existing form or create a new one by clicking on Add New in the WordPress admin menu
1. Add your fields
1. Add a text field named `your-name`
1. Add an email field named `your-email`
1. Add a SendPress Signup field named `spnl`
1. When you're adding the SendPress Signup field you can select any number of lists you want the user to be assigned to
1. You can also choose to make the user opt in or opt out
